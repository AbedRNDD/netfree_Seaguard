# خريطة الشباك المفقودة - Ghost Net Map

## التحديثات

تم دمج الخريطة التفاعلية مباشرة في صفحة Dashboard الرئيسية!

## الميزات

### 1. الخريطة التفاعلية (الصفحة الرئيسية)
- خريطة OpenStreetMap تعرض مواقع الشباك المفقودة مباشرة بعد تسجيل الدخول
- مركزة على منطقة الخليج العربي (الإحداثيات: 25.0, 45.0)
- 6 علامات حمراء تظهر مواقع الشباك المفقودة
- عند الضغط على العلامة، تظهر نافذة منبثقة بمعلومات الشبكة:
  - اسم الشبكة
  - اسم المستخدم الذي أبلغ عنها
  - تاريخ الإبلاغ

### 2. أدوات التحكم
- زر تكبير (+) - عائم على الجانب الأيمن
- زر تصغير (-) - عائم على الجانب الأيمن
- الأزرار تعمل بسلاسة مع الخريطة

### 3. تحسينات الأداء
- استخدام Canvas Renderer لتحسين الأداء
- تحميل العلامات بشكل متأخر (lazy loading)
- تحسين إعدادات التحديث للخريطة
- إزالة التأخير والتعليق

### 4. التنقل
- شريط علوي يعرض شعار التطبيق
- شريط سفلي للتنقل بين الصفحات:
  - الخريطة (الصفحة الحالية - نشطة)
  - بلاغ
  - تنبيهات
  - إحصائيات
- بطاقة إحصائيات أسفل الخريطة تعرض:
  - إجمالي الشباك: 6
  - النشطة: 1.2k
  - هذا الشهر: 89

## الملفات المعدلة

1. `src/main/resources/com/example/netfreeseaguard/dashboard.fxml` - دمج WebView للخريطة
2. `src/main/java/com/example/netfreeseaguard/DashboardController.java` - استخدام WebEngine
3. `src/main/resources/com/example/netfreeseaguard/map.html` - تحسينات الأداء والعلامات

## كيفية الاستخدام

1. شغل التطبيق
2. سجل الدخول
3. ستظهر الخريطة مباشرة كصفحة رئيسية
4. استخدم أزرار + و - للتكبير والتصغير
5. اضغط على العلامات الحمراء لرؤية تفاصيل الشباك المفقودة
6. استخدم شريط التنقل السفلي للانتقال بين الصفحات

## التطوير المستقبلي

يمكن إضافة:
- ربط الخريطة بقاعدة بيانات حقيقية
- إضافة فلاتر للبحث (حسب التاريخ، المستخدم، الحالة)
- إضافة أنواع مختلفة من العلامات (حسب حالة الشبكة)
- إمكانية إضافة شبكة جديدة من الخريطة مباشرة
- تتبع الموقع الحالي للمستخدم
- رسم مناطق خطرة على الخريطة
